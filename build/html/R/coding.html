2. 数据集说明¶

TOTAL VALUE

房产估价总值，单位：千美元

TAX

税单金额根据总评估价值乘以税率，单位：美元

LOT SQ FT

总平方英尺数

YR BUILT

房产建成年份

GROSS AREA

总建筑面积

LIVING AREA

房产的总居住面积

FLOORS

楼层数

ROOMS

房间数

BEDROOMS

卧室数

FULL BATH

全浴总数

HALF BATH

半浴总数

KITCHEN

厨房总数

FIREPLACE

壁炉总数

REMODEL

何时重装修的？(Recent/Old/None)
